📦 Step. 1 : HTML 필수 태그와 속성(특징)

  • html의 구조 (html, head, body)
  • 들여쓰는게 무슨 의미일까? (부모-자식)
  • 태그의 사용법 (닫는 태그 있어야 하나?)
  • <head>에 들어가는 태그들 (meta, title, link ...)
  • 반드시 알아두어야 할 태그들
  1. <div>, <span> (차이점)
  2. <p>, <br> (줄바꿈 vs 문단)
  3. <a> (href 속성 필수, 기본값 "#")
  4. <img /> (src 필수, 닫는 태그 없음)
  5. <input /> (type 속성에 따라 기능 다름)
  6. <button> (클릭 이벤트 처리)
  7. <table>, <tr>, <td> (표 구조)
  8. <ol>, <ul>, <li> (순서 있는/없는 목록)

📂 Step. 2 : 파일 경로

  • "/" → 루트 디렉토리
  • "./" → 현재 디렉토리
  • "../" → 상위 디렉토리

📱 Step. 3 : CSS 문법 & 선택자

HTML 태그에 style 속성으로 직접 스타일 적용 가능
.myClass { color: blue; }
  • id 선택자 → `#id`
  • class 선택자 → `.className`

💡 Step. 4 : 웹사이트 분석

개발자 도구(F12)를 이용해 화면 구조를 분석해보세요!

🔧 Step. 5 : 부트스트랩 적용

CDN 방식으로 부트스트랩을 연결합니다.


<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
