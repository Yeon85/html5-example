🏷️ 반드시 알아두어야 할 태그들

  • <h1> ~ <h6>: 제목 태그, 숫자가 작을수록 글씨가 큼
  • <div> vs <span>: 블록 vs 인라인
  • <p> vs <br>: 문단 vs 줄바꿈 / <p>는 <div>와 달리 의미 있는 문단
  • <a href="#">: 하이퍼링크, href 필수
  • <img src="경로" />: 이미지 태그 (닫는 태그 없음)
  • <input type="text" />: 사용자 입력 필드, 닫는 태그 없음
  • <button>: 버튼 생성
  • <table>, <tr>, <td>: 표 구성
  • <ol>, <ul>, <li>: 순서 있는/없는 목록 구성

📝 연습문제 + 풀이

  • ① <div>
  • ② <h1>
  • ③ <span> ✅
  • ④ <p>
