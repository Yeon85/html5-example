🔧 Step 5: CDN & Bootstrap

📡 CDN이란?

CDN(Content Delivery Network)은 전 세계 여러 서버에 분산된 파일(이미지, CSS, JS 등)을 빠르게 다운로드할 수 있게 해주는 기술입니다.

  • 웹사이트 로딩 속도 향상
  • 트래픽 분산
  • 전 세계 어디서든 빠르게 접근 가능
🛠️ CDN 적용 방법

CDN을 사용하면 <link> 또는 <script> 태그로 외부 파일을 가져올 수 있어요.

예: Bootstrap의 CSS 파일을 불러올 경우

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
🎨 부트스트랩이란?

Bootstrap은 HTML, CSS, JavaScript 기반의 웹 UI 프레임워크입니다.

  • 반응형 웹을 빠르게 만들 수 있음
  • 버튼, 카드, 모달 등 다양한 컴포넌트 제공
  • CSS 유틸리티 클래스가 풍부함
⚙️ 내 웹사이트에 부트스트랩 적용하기

CDN으로 부트스트랩을 사용하는 가장 쉬운 방법은 <head>에 링크를 추가하는 것입니다.

1. CSS 링크
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
2. JS (인터랙션 기능 사용 시 필수)
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
3. 실제 사용 예시
 
<button class="btn btn-primary">부트스트랩 버튼</button>
