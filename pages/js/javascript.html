JavaScript 기초입문

JavaScript란?

Java ≠ JavaScript ❌

JavaScript는 HTML, CSS와 함께 웹 개발의 핵심 요소입니다.

JavaScript는 프론트엔드(화면), 백엔드(서버, DB출력) 모두에서 사용됩니다. 예: Node.js, React.js, Angular.js

🧱 HTML, CSS, JavaScript 그림 설명

  • HTML: 건물의 뼈대 🏛 – 웹사이트의 구조를 만든다 (무엇이 있는지)
  • CSS: 건물의 색칠과 장식 🎀 – 웹사이트를 예쁘게 꾸민다 (디자인, 스타일)
  • JavaScript: 자동문과 엘리베이터 🛗 – 웹사이트를 움직이게 만든다 (동작, 반응)

🔢 1.변수 (Variables)

데이터를 저장하는 공간입니다.

  • let: 재할당 가능, 블록 스코프
  • const: 재할당 불가, 블록 스코프
  • var: ES6 이전 방식

⚡ 2.함수 (Functions)

특정 작업을 수행하는 코드 묶음입니다.

function add(x, y) { let temp = x + y; return temp; } const multiply = function(a, b) { return a * b; } console.log(4 * 5);

⚖️ 3.조건문 (Conditional Statements)

조건에 따라 흐름을 제어합니다.

let score = 85; if (score >= 90) { console.log("A 학점입니다."); } else if (score >= 80) { console.log("B 학점입니다."); } else if (score >= 70) { console.log("C 학점입니다."); } else if (score >= 60) { console.log("D 학점입니다."); } else { console.log("F 학점입니다."); }

🔁 4. 반복문 (Loops)

같은 작업을 반복합니다.

예: for 반복문

for (let i = 0; i < 5; i++) { console.log("i의 값은: " + i); }

위 예제는 i가 0부터 4까지 1씩 증가하며 반복되는 코드입니다.

🧺 5. 배열 (Arrays)

여러 데이터를 하나로 묶습니다.

예제:

const fruits = ["사과", "바나나", "키위"]; // 값 추가 fruits.push("포도"); // ["사과", "바나나", "키위", "포도"] // 값 제거 fruits.pop(); // ["사과", "바나나", "키위"] // 탐색 (하나씩 출력) fruits.forEach(function(fruit) { console.log(fruit); });

위 예제에서 push()는 배열 끝에 값을 추가하고, pop()은 마지막 값을 제거합니다.

forEach()는 배열의 각 항목을 하나씩 꺼내서 처리합니다.
