📘 Step. 3 : CSS

✅ CSS 적용 안될 경우?

✔ HTML의 <head> 태그에서 CSS 파일이 연결되어 있는지 확인!
예: <link rel="stylesheet" href="style.css">

✅ CSS 기본 문법

div {
  color: blue;
}
  • { } 중괄호 안에 스타일 작성
  • ;는 각 속성 선언의 끝에 붙임

✅ CSS 적용 방법

  • HTML 태그에 직접 적용: <div style="color:blue">
  • ID 선택자 (#): #myId { ... }
  • Class 선택자 (.): .myClass { ... }

✅ ID vs CLASS 차이

  • ID: 문서 내에서 한 번만 사용
  • CLASS: 여러 요소에 반복 사용 가능

✅ 선택자 종류

  • 복합선택자: 여러 조건 조합
  • 일치선택자: div.style { }
  • 자식선택자: div > .style { }
  • 후손선택자: div .style { }

✅ 자주 쓰이는 스타일 속성

  • color: 글자 색
  • background-color: 배경 색
  • border / border-radius: 테두리 / 둥글게
  • solid: 테두리 실선
  • width / height: 너비 / 높이
  • display: flex: 레이아웃 정렬
  • justify-content: 주축 정렬
  • align-items: 교차축 정렬
  • flex-direction: 방향 설정
  • font-family: 글꼴 지정
  • text-align: 글자 정렬
  • margin / padding: 바깥 여백 / 안쪽 여백
  • float: 요소를 띄움
  • text-decoration: 밑줄, 취소선 등

✅ 가상 클래스

  • :hover: 마우스를 올렸을 때 효과

✅ position 속성

  • static: 기본값 (기준 없음)
  • relative: 자기 기준 위치 이동
  • absolute: 가장 가까운 부모 기준
  • fixed: 화면 기준 (스크롤에도 고정)
  • sticky: 스크롤하다가 특정 위치에서 고정

✅ 단위 종류

  • px: 픽셀
  • in: 인치
  • %: 상대 비율
  • vh: 화면 높이 기준 비율

✅ 미디어쿼리 (반응형 웹)

@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}
