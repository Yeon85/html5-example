💡 부트스트랩(Bootstrap)이란?

웹 UI를 빠르게 만들 수 있도록 도와주는 프론트엔드 프레임워크

✅ 핵심 요약

HTML, CSS, JS로 구성된 웹 UI 템플릿 모음집으로, 반응형 웹사이트를 빠르게 만들 수 있어요.

👨‍💻 누가 만들었나요?

트위터(Twitter)의 개발자들이 내부 도구로 만들었고, 지금은 전 세계 개발자들이 사용하는 오픈소스입니다.

✨ 왜 사용할까요?

  • 빠른 개발: 이미 만들어진 UI 요소들을 그대로 사용
  • 반응형 디자인: 다양한 화면 크기에 자동 대응
  • 일관된 스타일: 디자인이 통일감 있게 구성됨
  • 배우기 쉬움: 문서화가 잘 되어 있음

🧱 어떤 기능이 있나요?

  1. 그리드 시스템 (12칸 레이아웃)
  2. 다양한 UI 컴포넌트 (버튼, 모달, 카드 등)
  3. 유틸리티 클래스 (빠르게 스타일 적용)

🔧 예시 코드

아래는 부트스트랩으로 만든 버튼 예시입니다:

<button class="btn btn-primary">버튼</button>
